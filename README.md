# juanschallmoser.dev

Personal portfolio — Backend Engineer.

🔗 **Live:** [schallmoserjuan.dev/](https://schallmoserjuan.pages.dev/)

## Stack

- **Frontend:** HTML · Tailwind CSS v4 · Vanilla JS
- **Form:** [Formspree](https://formspree.io)
- **Hosting:** Cloudflare Pages

## Features

- Dark / light mode with persistence
- EN / ES language toggle (i18n)
- Scroll reveal animations
- Parallax grid background
- Working contact form (Formspree)
- SEO meta tags + Open Graph
- Fully responsive

## Development

```bash
# Install dependencies
npm install

# Watch mode (auto-recompile CSS on change)
npm run watch:css

# Production build
npm run build:css
```
